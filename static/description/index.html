Dynamic Field Access Control

Take full control of your Odoo fields with dynamic, per-user or per-group access. Configure fields to be readonly or invisible without touching code.

Key Features

  • Select Target Models: Apply field access rules to any model in Odoo.
  • User & Group Based Rules: Restrict access for specific users or groups.
  • Prevent All Updates and Deletes: Protect critical records from modification or removal.
  • Field-Specific Restrictions: Make only selected fields readonly or hidden.
  • Usage-Aware Restrictions: Prevent updates, deletes, or duplication based on record usage.

How It Works

Administrators can configure access rules through a simple UI. The rules are applied dynamically depending on the logged-in user or group, ensuring data integrity and secure operation of business-critical objects.
